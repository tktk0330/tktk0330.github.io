トモッ帳 プライバシーポリシー

最終更新日: 2026年3月20日

本プライバシーポリシー（以下「本ポリシー」）は、トモッ帳（以下「本アプリ」）における個人情報およびプライバシー情報の取り扱いについて説明するものです。お子さまのプライバシー保護を最優先に設計されています。

保護者の方へ: お子さまが本アプリを利用される前に、本ポリシーをお読みいただき、内容をご理解ください。

1. 運営者情報

本アプリの運営者（以下「当方」）に関するお問い合わせは、本アプリ内の「せってい」画面に記載のお問い合わせ先までご連絡ください。

2. 適用地域

本アプリは日本国内での利用を前提として提供されており、日本の個人情報保護法その他の関連法令を遵守します。

3. 児童のプライバシー保護について

  1. 本アプリは、お子さまから必要最小限の情報のみを収集するよう設計されています。
  2. メールアドレス、パスワード、本名、住所、電話番号等の個人を特定できる情報の入力を求めることはありません。
  3. 広告の表示は、児童の安全に配慮した設定（児童向け指定、一般向けコンテンツレーティング）で行われます。
  4. 外部リンク（本ポリシーやFAQ等）へのアクセスには、保護者認証（算数問題による認証ゲート）が設けられています。

4. 収集する情報

本アプリが収集・保存する情報は以下のとおりです。

プロフィールカードに記載された内容について: 利用者がプロフィールカードに記入した手書きデータ、装飾データ、スキャン画像等のコンテンツは、利用者のデバイス内（および iCloud 同期先のデバイス）にのみ保存されます。これらのデータが当方のサーバーに送信・収集されることはありません。プロフィール交換時も、デバイス間の直接通信（Bluetooth / Wi-Fi）で送受信され、当方のサーバーを経由しません。

4-1. 利用者が入力し、当方が収集する情報

情報の種類 内容 目的
ニックネーム 利用者が設定する表示名（1〜10文字） プロフィール交換時の表示名として使用

4-2. デバイス内にのみ保存される情報（当方は収集しません）

情報の種類 内容 保存場所
プロフィールカードデータ 手書きデータ（PencilKit描画データ）、装飾データ デバイス内 / iCloud
スキャン画像 カメラでスキャンした紙のプロフィール帳の画像（JPEG） デバイス内 / iCloud

4-3. 自動的に収集される情報

情報の種類 内容 目的
匿名識別子 Firebase Anonymous Authentication により自動生成される匿名 UID アプリ管理（メンテナンス通知・強制アップデート判定等）

プロフィール交換は Bluetooth および Wi-Fi を用いた近距離通信（Apple Multipeer Connectivity）で行われ、当方のサーバーを経由しません。交換データはデバイス間で直接送受信されます。

5. 情報の保存場所

5-1. デバイス内（ローカルストレージ）

プロフィールカードデータ、交換履歴、アクティビティ履歴等のデータは、利用者のデバイス内に保存されます。

5-2. iCloud（Apple CloudKit）

同じ Apple ID でサインインしている複数のデバイス間で、データが自動的に同期されます。この同期は Apple の iCloud サービスを利用しており、データは Apple のプライバシーポリシーに基づいて管理されます。

5-3. Firebase（Google Cloud）

以下の情報が Firebase サービス上に保存されます。

  • 匿名認証情報（Firebase Authentication）
  • ニックネーム、デバイス情報、アプリバージョン、タイムスタンプ（Cloud Firestore）

Firebase に保存されるデータにはメールアドレス、本名、住所等の個人を直接特定できる情報は含まれません。

6. 収集する情報の最小化について

当方は、本アプリの運営に必要最小限の情報のみを収集しています。利用者がプロフィールカードに記載した内容（手書きデータ、スキャン画像、装飾データ等）は当方のサーバーには一切送信・保存されず、当方がその内容を閲覧・取得することはできません。

7. 情報の利用目的

収集した情報は、以下の目的でのみ利用します。

  1. 本アプリの機能提供（プロフィールカードの作成・交換・閲覧）
  2. 複数デバイス間のデータ同期
  3. アプリの強制アップデート・メンテナンス通知
  4. アプリの不具合調査・改善
  5. 返却きげんのリマインダー通知（ローカル通知）
  6. 広告の表示（第9条に定めるとおり）

8. 情報の第三者提供

  1. 当方は、法令に基づく場合を除き、利用者の個人情報を第三者に提供しません。
  2. ただし、以下のサービス提供者がアプリの運営に必要な範囲で情報にアクセスする場合があります。
サービス提供者 サービス名 利用目的 プライバシーポリシー
Apple Inc. iCloud (CloudKit) デバイス間データ同期 Apple プライバシーポリシー
Google LLC Firebase Authentication 匿名ユーザー認証 Google プライバシーポリシー
Google LLC Cloud Firestore ユーザーメタデータ保存
Google LLC Google AdMob 広告配信

9. 広告について

  1. 本アプリでは、Google AdMob を利用した広告が表示される場合があります。
  2. 広告配信にあたり、以下のプライバシー保護設定を適用しています。
    • 児童向けコンテンツとして設定（児童向け指定を有効化）
    • 年齢に配慮した広告配信設定を有効化
    • コンテンツレーティング: 一般（G）に制限
  3. パーソナライズされた広告を表示するためには、Apple の App Tracking Transparency（ATT）フレームワークを通じた利用者の明示的な許可が必要です。
  4. ATT で許可しなかった場合、広告識別子（IDFA）は収集されず、パーソナライズされていない広告のみが表示されます。
  5. ATT の許可状態は、デバイスの「設定」アプリからいつでも変更できます。

10. アナリティクスについて

  1. 本アプリでは、サービス改善のためにアプリ内イベント（画面表示、機能利用等）を記録する場合があります。
  2. 記録されるイベントには、個人を特定できる情報は含まれません。
  3. イベントデータはアプリの利用状況を統計的に分析するためにのみ使用されます。

11. 通知について

  1. 本アプリは、返却きげんのリマインダーとしてローカル通知を使用します。
  2. 通知はデバイス上でスケジュールされ、外部サーバーを経由しません。
  3. 通知の受信は、デバイスの「設定」アプリからいつでも無効にできます。

12. データの安全管理措置

当方は、利用者の情報を適切に保護するため、以下の安全管理措置を講じています。

  1. 通信の保護: Firebase との通信は TLS（SSL）で暗号化されます。近距離交換は Apple Multipeer Connectivity の暗号化機能を利用します。
  2. 画面キャプチャ保護: プロフィールカードの内容がスクリーンショットや画面録画で取得されることを防止する機能を搭載しています。
  3. NGワードフィルター: 不適切な言葉の入力を検出・マスクするフィルター機能を搭載しています。
  4. 保護者認証: 外部リンクへのアクセスに算数問題による認証ゲートを設けています。
  5. データ保存: ローカルデータは Apple の標準的なデータ保護（Data Protection）により暗号化されます。

13. 利用者の権利

利用者（または保護者）は、以下の権利を有します。

  1. データの確認: 本アプリ内で保存されているデータは、アプリの各画面から確認できます。
  2. データの削除: アプリを削除することで、デバイス内のすべてのデータを削除できます。iCloud に同期されたデータは、デバイスの「設定」>「Apple ID」>「iCloud」から管理・削除できます。
  3. Firebase データの削除: Firebase 上に保存された匿名アカウント情報およびメタデータの削除をご希望の場合は、お問い合わせ先までご連絡ください。
  4. 広告パーソナライズの制御: デバイスの「設定」>「プライバシーとセキュリティ」>「トラッキング」から、広告のパーソナライズ許可をいつでも変更できます。
  5. 通知の制御: デバイスの「設定」から通知の許可をいつでも変更できます。

14. カメラの使用について

  1. 本アプリは、紙のプロフィール帳をスキャンして取り込むためにカメラを使用します。
  2. カメラで撮影した画像は、デバイス内に保存され、プロフィールカードの一部としてのみ使用されます。
  3. 撮影した画像が当方のサーバーに自動送信されることはありません。
  4. カメラへのアクセス許可は、デバイスの「設定」アプリからいつでも取り消すことができます。

15. Bluetooth / ローカルネットワークの使用について

  1. 本アプリは、近くの友だちとプロフィールカードを交換するために Bluetooth およびローカルネットワーク（Wi-Fi）を使用します。
  2. これらの通信は、プロフィール交換機能を利用する場合にのみ使用されます。
  3. 位置情報の取得には使用しません。
  4. アクセス許可は、デバイスの「設定」アプリからいつでも取り消すことができます。

16. データの保存期間

  1. デバイス内データ: アプリが削除されるまで保存されます。
  2. iCloud データ: 利用者が iCloud から削除するまで保存されます。
  3. Firebase データ: アカウント削除のご依頼があるまで保存されます。長期間ご利用のない匿名アカウントは、Firebase の自動クリーンアップにより削除される場合があります。

17. 海外へのデータ移転

本アプリは日本国内向けに提供していますが、Firebase（Google Cloud）のサーバーは日本国外に所在する場合があります。これに伴い、第5条3項に記載のデータが日本国外に移転される場合がありますが、Google のプライバシーポリシーおよびセキュリティ基準に基づいて保護されます。
